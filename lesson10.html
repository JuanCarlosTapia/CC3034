Linters de javascript

JSLint

  • El primer linter de javascript que se popularizó
  • 2002 - Douglas Crockford (Javascript the good parts)
  • Enforza las ideas personales de su autor

Ventajas

  • Viene pre-empacado y listo para usar

Desventajas

  • No se puede personalizar
  • No se puede utilizar para el mal
  • https://en.wikipedia.org/wiki/Douglas_Crockford
  • http://dev.hasenj.org/post/3272592502/ibm-and-its-minions
http://jslint.com/

JSHint

  • Nace como un fork de JSLint
  • Se enfoca en ser mucho mas configurable

Ventajas

  • Se puede configurar considerablemente
  • Integra soporte de varias librerias de javacript comunes
  • Soporte de javascript 6

Desventajas

  • La configuración por default es muy permisiva
  • Es un poco complejo al configurar
  • No permite reglas custom
http://jshint.com/

ESLint

  • 2013 - Nicolas Zakas
  • Se desarrolla con el objetivo que todo sea pluggeable

Ventajas

  • Set robusto de reglas
  • Facilmente personalizable
  • Soporte de javascript 6

Desventajas

  • La configuración por default es muy permisiva
  • Es un poco complejo al configurar
  • No permite reglas custom
